# vue3.0 micro 模板
> ant 在vue3.0 中以一定的问题，暂时选择使用 element-puls

## 安装依赖

> `yarn of npm i`

## 运行项目

> `yarn serve`

## 将代码添加到缓存区

> `yarn gd`

## 添加 commit

> `yarn commit`

## 推送代码到远程

> `yarn push branchName`
